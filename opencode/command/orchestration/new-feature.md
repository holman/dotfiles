Orchestrate complete feature development from architecture to production readiness.

Workflow Stages:

1. **Architecture Design** (Master Architect)
   - System design and component architecture
   - Technology stack recommendations
   - Integration points and dependencies
   - Scalability and performance considerations

2. **Implementation** (Frontend/Backend/Mobile Engineers)
   - Frontend components with React/Next.js
   - Backend APIs and services
   - Mobile features if applicable
   - Database schema changes

3. **Security Review** (Security Auditor)
   - Vulnerability assessment
   - Authentication/authorization review
   - OWASP Top 10 compliance
   - Security control recommendations

4. **Performance Validation** (Performance Analyst)
   - Performance impact assessment
   - Optimization recommendations
   - Load testing strategy
   - Caching implementation

5. **Accessibility Compliance** (Accessibility Engineer)
   - WCAG 2.1 AA compliance verification
   - Screen reader testing
   - Keyboard navigation validation
   - Remediation guidance

6. **Test Coverage** (Test Automation Engineer)
   - Unit test implementation
   - Integration test creation
   - E2E test scenarios
   - Test coverage analysis

7. **Code Quality** (Code Reviewer)
   - Code review and quality assessment
   - Best practices validation
   - Refactoring recommendations

8. **Deployment Planning** (Deployment Engineer)
   - CI/CD pipeline configuration
   - Deployment strategy
   - Rollback plan
   - Production monitoring

Provide comprehensive feature delivery with all quality gates passed.
