# Orchestration Commands

Commands for workflow management, coordination, and process optimization.

## Commands

### [build](build.md)
Build process management and optimization.

### [hotfix](hotfix.md)
Emergency fixes and rapid deployment coordination.

### [invoke](invoke.md)
Command invocation and execution management.

### [new-feature](new-feature.md)
New feature development coordination and planning.

### [plan](plan.md)
Project planning and task organization.

## Usage

Commands in this category focus on:
- **Workflow Management**: Process coordination and optimization
- **Build Management**: Build pipeline design and execution
- **Release Management**: Deployment coordination and versioning
- **Task Coordination**: Multi-agent workflow orchestration

## Common Scenarios

- Coordinating new feature development
- Managing build and deployment processes
- Handling emergency fixes and hotfixes
- Planning project roadmaps and timelines
- Optimizing development workflows

## Related Commands

- `/analyze` - Analysis for planning decisions
- `/review` - Review for build and deployment readiness
- `/optimize` - Workflow and process optimization
