# Meta Commands

Meta-level commands for session management, analysis, and coordination tasks.

## Commands

### [session-summary](session-summary.md)
Session summary and documentation of work completed.

## Usage

Commands in this category focus on:
- **Session Management**: Tracking and documenting work sessions
- **Coordination**: Meta-level task organization
- **Documentation**: High-level documentation and summaries
- **Analysis**: Meta-analysis of work and progress

## Common Scenarios

- Summarizing development sessions
- Documenting progress and decisions
- Coordinating multi-agent workflows
- Analyzing development patterns
- Creating progress reports

## Related Commands

- `/analyze` - Detailed code analysis
- `/review` - Comprehensive code review
- `/plan` - Project planning and organization
