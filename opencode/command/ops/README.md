# Operations Commands

Commands for deployment, performance, security, and infrastructure management.

## Commands

### [optimize](optimize.md)
Performance optimization and efficiency improvements.

### [performance-audit](performance-audit.md)
Comprehensive performance analysis and bottleneck identification.

### [security-audit](security-audit.md)
Security vulnerability assessment and best practices review.

## Usage

Commands in this category focus on:
- **Performance**: Optimization, monitoring, and efficiency
- **Security**: Vulnerability assessment and best practices
- **Infrastructure**: Deployment and operational concerns
- **Monitoring**: Observability and alerting

## Common Scenarios

- Performance optimization and profiling
- Security audits and vulnerability assessment
- Infrastructure deployment and management
- Monitoring and alerting setup
- Operational best practices implementation

## Related Commands

- `/analyze` - Context-aware analysis including operational concerns
- `/review` - Code review with operational considerations
- `/refactor` - Performance-focused refactoring
