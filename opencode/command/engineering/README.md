# Engineering Commands

Commands for implementation tasks across different platforms and technologies.

## Commands

### [be](be.md)
Backend development tasks and implementation.

### [fe](fe.md)
Frontend development tasks and UI implementation.

### [mobile](mobile.md)
Mobile development tasks and platform-specific implementation.

## Usage

Commands in this category focus on:
- **Implementation**: Code development and feature implementation
- **Platform-Specific**: Technology and framework-specific tasks
- **Best Practices**: Coding standards and patterns
- **Integration**: Component and system integration

## Common Scenarios

- Implementing new features
- Refactoring existing code
- Platform-specific optimizations
- Integration development
- Code maintenance and updates

## Related Commands

- `/analyze` - Context-aware code analysis
- `/review` - Comprehensive code review
- `/refactor` - Code restructuring and improvement
- `/improve` - Code enhancement and optimization
