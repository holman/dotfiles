# Architecture Commands

Commands for system design, architectural planning, and technology decisions.

## Commands

### [arch](arch.md)
System architecture analysis and design recommendations.

## Usage

Commands in this category focus on:
- **System Design**: Architecture patterns and best practices
- **Technology Selection**: Framework and infrastructure decisions
- **Scalability Planning**: Performance and growth considerations
- **Integration Strategy**: Service communication and data flow

## Common Scenarios

- Designing new system architectures
- Evaluating technology stack choices
- Planning system scalability
- Reviewing existing architectures
- Planning system integrations

## Related Commands

- `/plan` - Feature planning and roadmapping
- `/analyze` - Context-aware code analysis
- `/review` - Comprehensive code review
