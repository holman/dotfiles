# Engineering Agents

Specialized AI agents focused on implementation across different platforms and technologies.

## Agents

### [Frontend Engineer](frontend-engineer.md)
Expert in React, TypeScript, CSS, state management, and modern web development practices.

### [Mobile Engineer](mobile-engineer.md)
Specializes in React Native, Swift, Kotlin, and mobile-specific development patterns.

## Expertise Areas

- **Frontend Development**: React, Vue, Angular, modern JavaScript/TypeScript
- **Mobile Development**: iOS, Android, React Native, Flutter
- **State Management**: Redux, MobX, Context API, mobile state patterns
- **UI/UX Implementation**: Responsive design, accessibility, performance
- **Build Systems**: Webpack, Vite, mobile build pipelines
- **Testing**: Unit tests, integration tests, E2E testing

## Common Tasks

- Component development and architecture
- Performance optimization
- Code review and best practices
- Build system configuration
- Testing strategy and implementation
- Cross-platform compatibility

## Collaboration

Engineering agents often work with:
- **Architecture agents** for design decisions
- **Quality agents** for testing and code review
- **Operations agents** for deployment and performance
