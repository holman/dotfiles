# Operations Agents

Specialized AI agents focused on deployment, performance, security, and infrastructure management.

## Agents

### [Deployment Engineer](deployment-engineer.md)
Expert in CI/CD pipelines, Docker, cloud deployment, and infrastructure as code.

### [Performance Analyst](performance-analyst.md)
Specializes in optimization, profiling, bottleneck identification, and performance monitoring.

### [Security Auditor](security-auditor.md)
Focuses on vulnerability assessment, security best practices, and compliance requirements.

## Expertise Areas

- **DevOps**: CI/CD pipelines, automation, deployment strategies
- **Cloud Infrastructure**: AWS, Azure, GCP, container orchestration
- **Performance Optimization**: Profiling, monitoring, bottleneck analysis
- **Security**: Vulnerability assessment, compliance, best practices
- **Monitoring**: Observability, logging, alerting, metrics
- **Infrastructure as Code**: Terraform, CloudFormation, Ansible

## Common Tasks

- Deployment pipeline design and implementation
- Performance optimization and monitoring
- Security audits and vulnerability assessments
- Infrastructure provisioning and management
- Monitoring and alerting setup
- Disaster recovery planning

## Collaboration

Operations agents often work with:
- **Engineering agents** for deployment readiness
- **Architecture agents** for infrastructure design
- **Quality agents** for security and performance testing
