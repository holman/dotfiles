# Orchestration Agents

Specialized AI agents focused on workflow management, coordination, and process optimization.

## Agents

### [Workflow Orchestrator](workflow-orchestrator.md)
Expert in build processes, release management, task coordination, and workflow automation.

## Expertise Areas

- **Build Management**: Build pipeline design, dependency management, build optimization
- **Release Management**: Version control, release planning, deployment coordination
- **Workflow Automation**: Process automation, task scheduling, integration management
- **Project Coordination**: Task tracking, resource allocation, timeline management
- **Process Optimization**: Workflow analysis, efficiency improvements, bottleneck identification
- **Integration Management**: System integration, API coordination, data flow optimization

## Common Tasks

- Build pipeline design and optimization
- Release planning and coordination
- Workflow automation implementation
- Process analysis and improvement
- Task scheduling and management
- Integration strategy development

## Collaboration

Orchestration agents often work with:
- **Engineering agents** for build and deployment processes
- **Operations agents** for infrastructure and deployment
- **Architecture agents** for system integration
- **Quality agents** for testing and validation workflows
