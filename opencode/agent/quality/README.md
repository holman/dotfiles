# Quality Agents

Specialized AI agents focused on code quality, testing, accessibility, and standards compliance.

## Agents

### [Accessibility Engineer](accessibility-engineer.md)
Expert in a11y compliance, screen readers, inclusive design, and accessibility standards.

### [Code Reviewer](code-reviewer.md)
Specializes in code quality, maintainability, best practices, and architectural compliance.

### [Test Automation Engineer](test-automation-engineer.md)
Focuses on test strategy, coverage, automation frameworks, and quality assurance.

## Expertise Areas

- **Code Quality**: Best practices, maintainability, code review standards
- **Testing**: Unit tests, integration tests, E2E testing, test automation
- **Accessibility**: WCAG compliance, screen readers, inclusive design
- **Quality Assurance**: Quality gates, standards compliance, metrics
- **Performance Testing**: Load testing, performance benchmarks
- **Security Testing**: Vulnerability scanning, security validation

## Common Tasks

- Comprehensive code reviews
- Test strategy development and implementation
- Accessibility audits and improvements
- Quality standards definition and enforcement
- Performance and security testing
- Quality metrics and reporting

## Collaboration

Quality agents often work with:
- **Engineering agents** for implementation quality
- **Architecture agents** for design quality
- **Operations agents** for deployment and performance quality
