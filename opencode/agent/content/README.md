# Content Agents

Specialized AI agents focused on technical writing, documentation, and content creation.

## Agents

### [Content Editor](content-editor.md)
Expert in technical writing clarity, content structure, flow, and editorial standards.

### [Documentation Specialist](documentation-specialist.md)
Specializes in API documentation, user guides, technical specifications, and documentation architecture.

## Expertise Areas

- **Technical Writing**: Clear, concise, and accurate technical content
- **Documentation Architecture**: Structured documentation systems and information architecture
- **Content Strategy**: Planning, organizing, and maintaining technical content
- **Editorial Standards**: Style guides, consistency, and quality assurance
- **User Experience**: User-focused documentation and help systems
- **API Documentation**: Comprehensive API references and examples

## Common Tasks

- Technical documentation creation and maintenance
- Content review and editing
- Documentation architecture design
- Style guide development and enforcement
- User guide and tutorial creation
- API reference documentation

## Collaboration

Content agents often work with:
- **Engineering agents** for technical accuracy
- **Architecture agents** for system documentation
- **Quality agents** for accessibility and usability standards
